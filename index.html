Como criar um curso

Acessando o ambiente como Admin, você pode criar um curso utilizando a plataforma Tim Tec. Para isso:

  1. Selecione o menu Cursos.
  2. Clique no botão NOVO CURSO. A tela exibida é a tela inicial do seu curso.
  3. Clique em cada um dos campos para inserir os dados de seu curso e salve cada alteração. Veja aqui como preencher cada campo.
  4. Salve seu curso clicando no botão Salvar. Veja que aqui, o curso ainda não está visível para o público, mas sim apenas para você.
  5. Torne seu curso visível ao público na plataforma. Para isso, clique no botão Publicar.

Como preencher a página inicial do curso

O preenchimento de todo campo é obrigatório. Caso algum campo não seja preenchido, ele será exibido vazio ao aluno.

Veja a seguir a descrição de cada um dos campos:

  1. Nome do curso: preencha com o nome de seu curso. Veja que abaixo deste campo, automaticamente é gerado o link direto para o curso, com o uso de hífens para a separação das palavras.
  2. Vídeo ou Thumbnail: aqui você escolhe se sua página inicial do curso terá um vídeo de apresentação ou uma imagem. Caso for um vídeo, clique em VIDEO, insira o seu link do YouTube e clique em Adicionar. Caso for uma imagem, clique em Thumbnail, escolha o arquivo de sua imagem e clique em Adicionar.
  3. Abstract: insira aqui um resumo apresentando seu curso. Nossa sugestão é que você use palavras-chave indicando sobre o que trata o curso.
  4. Estrutura: neste campo você deve explicar de maneira sucinta a metodologia do seu curso, como serão oferecidas as aulas, quais os tipos de atividade, avaliação, enfim, quais os recursos de aprendizagens disponíveis.
  5. Carga horária: preencha aqui com o tempo estimado de duração de seu curso. Um modo para isso é somar os tempos de videoaula e a estimativa de quanto o aluno gastaria com as atividades. Sugerimos também que você descreva ao aluno um tempo de dedicação semanal de estudo para que ele melhor aproveite o curso.
  6. PRONATEC: insira aqui as informações que relacionam seu curso ao Pronatec, como por exemplo, a que currículo este curso pode estar alinhado.
  7. Aplicação: descreva aqui de maneira clara quais são as aplicações dos conhecimentos adquiridos após a conclusão do cursos, nos contextos profissional e social do aluno. Pense nas seguintes perguntas: o que posso estudar agora? O que consigo fazer agora? Em que posso trabalhar agora?
  8. Requisitos: utilize este campo para descrever dois tipos de requisitos:
    • Quais os conhecimentos prévios necessários para realizar este curso.
    • Quais os requisitos de infraestrutura para realizar este curso.
  9. Lições: este campo contém o principal conteúdo de seu curso. Aqui, serão inseridas as videoaulas e os vídeos.

Como editar um curso?

Você sempre poderá editar os conteúdos dos cursos criados por você. Para isso:

  1. No menu principal, selecione a opção Admin.
  2. Entre no curso que deseja editar.
  3. Clique sobre o(s) campo(s) que deseja editar e realize as mudanças desejadas.
  4. Clique no botão Salvar.
  5. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como apagar um curso?

Para remover um curso, realize os seguintes passos:

  1. No menu principal, selecione a opção Admin.
  2. Entre no curso que deseja apagar.
  3. Clique no botão Apagar este curso.
  4. Confirme no aviso que você deseja remover o curso e pronto.

    5. Pronto! O curso e todo o conteúdo contido nela foram removidos da plataforma TIM Tec.

Como alterar o vídeo ou a imagem da tela inicial de seu curso?

Você pode a qualquer momento alterar o vídeo ou a imagem escolhida para a tela inicial do curso. Para isso:

  1. No menu principal, selecione a opção Admin.
  2. Entre no curso que deseja alterar.
  3. Para alterar o vídeo: clique em TROCAR VÍDEO, apague o link do vídeo atual do campo Vídeo introdutório, insira o novo link e clique em ADICIONAR.
    Para alterar a imagem (thumbnail): clique em TROCAR THUMBNAIL, busque a imagem através do botão Choose File e clique em ADICIONAR.
  4. Clique no botão Salvar.
  5. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como adicionar uma lição?

As lições são responsáveis por grande parte conteúdo do curso. Ela é constituída por unidades sendo que cada unidade contém um vídeo obrigatoriamente e pode conter quantas atividades você desejar.

Para adicionar uma lição, realize os seguintes passos:

  1. No menu principal, selecione a opção Admin.
  2. Entre no curso que deseja adicionar uma ou mais lições.
  3. Clique em ADICIONAR AULA. A tela de edição da aula abrirá.
  4. Defina um nome para essa aula. Nossa sugestão é que seja objetiva e de fácil entendimento.
  5. Insira a descrição de sua aula no campo que possui este mesmo nome. Esta descrição será exibida abaixo do nome da aula, na tela inicial do curso.
  6. Adicione quantas unidades desejar clicando na barra verde com o símbolo "+". Para cada unidade, é obrigatória a inserção de um vídeo. As atividades ficam a critério do professor. Veja os tipos de atividade disponíveis em Como adicionar uma atividade.
  7. Clique em ADICIONAR para cada vídeo ou atividade adicionado.
  8. Preencha o campo Notas com informações úteis ao aluno relacionadas a cada unidade. Como sugestão, você pode inserir dicas sobre as atividades ou conteúdo extra sobre a unidade.
  9. Clique no botão Salvar.
  10. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como editar uma lição?

Você pode alterar qualquer conteúdo de uma lição, desde seu título ou descrição, até suas atividades.

Para isso, realize os seguintes passos:

  1. Com acesso no papel de Admin, dentro do curso que deseja alterar, acesse a lição desejada.
  2. Clique sobre o campo que deseja alterar: título, descrição, unidadeo e nota.
  3. Realize as alterações desejadas.
  4. Clique no botão Salvar.
  5. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como apagar uma lição?

Você pode apagar qualquer lição dentro de um curso. Para isso:

  1. Com acesso no papel de Admin, dentro do curso que deseja alterar, acesse a lição desejada.
  2. Clique no botão Apagar esta aula.
  3. Clique em OK na caixa de confirmação que aparecerá.

Pronto! A lição e todo o conteúdo contido nela foram removidos do curso e da plataforma TIM Tec.

Como adicionar vídeos nas lições

Um lição pode possuir várias unidades e cada unidade possui um vídeo obrigatoriamente.

Para adicionar o vídeo:

  1. Com acesso no papel de Admin, dentro do curso que deseja alterar, acesse a lição em que deseja inserir o vídeo.
  2. Clique na unidade em que deseja inserir o vídeo. Veja que para adicionar unidades basta clicar na barra verde com o símbolo "+".
  3. Insira o link do YouTube no campo Vídeo da unidade e clique no botão ADICIONAR.
  4. Clique no botão Salvar.
  5. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como alterar o vídeo de uma lição

Para alterar um ou mais vídeo nas lições, realize os seguintes passos:

  1. Com acesso no papel de Admin, dentro do curso escolhido, acesse a lição em que deseja alterar o vídeo.
  2. Clique na unidade em que o vídeo encontra-se.
  3. Clique no botão TROCAR VÍDEO e insira o link do YouTube no campo Vídeo da unidade.
  4. Clique no botão Salvar.
  5. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como adicionar uma atividade

Dentro de cada unidade da lição, após ter inserido o vídeo, você pode inserir quantas atividades desejar.

Veja a seguir como inserir as atividades e quais são os tipos disponíveis na plataforma TIM Tec:

  1. Com acesso no papel de Admin, dentro do curso escolhido, acesse a lição em que deseja inserir uma ou mais atividades.
  2. Clique na unidade escolhida.
  3. Clique no botão ATIVIDADE.
  4. Para a primeira atividade, clique no 1 do menu numérico e navegação. Para adicionar mais atividades na mesma lição, clique no + deste mesmo menu.
  5. Escolha um dos tipos de atividade descritos brevemente a seguir:
    • Escolha simples: composta por um enunciado e alternativas, sendo apenas uma como escolha de resposta.
    • Múltipla escolha: composta por um enunciado e alternativas, podendo ser mais de uma como escolha de resposta. Veja que aqui o aluno deverá marcar todas as alternativas que você escolher.
    • Verdadeiro ou Falso: composta por enunciado e alternativas, sendo que para cada uma é necessário assinalar Verdadeiro ou Falso.
    • Relacionar sentenças: composta por enunciado, coluna 1 com alternativas e coluna 2 com textos para relacionar a cada uma dessas alternativas.
    • HTML5: composta por enunciado, campo de resultado esperado em editor HTML5 e editor do aluno. A exibição para o aluno do resultado é sempre a renderização do código inserido por você.
    • Texto simples: composta por enunciado. Na visão do aluno, aparecerá um campo de texto em branco para resposta.

    Para cada atividade, há um campo de comentário (Comment) disponível para ser exibido após o aluno responder a atividade.

  6. Clique no botão ADICIONAR.
  7. Preencha os campos necessários da atividades. Veja mais sobre cada atividade em Como preencher cada tipo de atividade.
  8. Clique no botão Salvar.
  9. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como editar uma atividade

Você pode editar os campos de qualquer atividade inserida. Para isso:

  1. Com acesso no papel de Admin, dentro do curso, acesse a lição em que a atividade escolhida encontra-se.
  2. Clique na unidade que contém a atividade.
  3. No menu numeŕico, clique no número referente à atividade que deseja alterar.
  4. Clique no(s) campo(s) que deseja modificar e realize suas alterações.
  5. Clique no botão Salvar.
  6. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como apagar uma atividade?

Você pode apagar qualquer atividade inserida, sem alterar o resto da unidade em que ela se encontra. Para isso:

  1. Com acesso no papel de Admin, dentro do curso, acesse a lição em que a atividade escolhida encontra-se.
  2. Clique na unidade que contém a atividade.
  3. Escolha no menu numérico qual a atividade a ser removida.
  4. Clique no botão Apagar esta atividade.

Pronto! A atividade foi removida de seu curso e da plataforma TIM Tec.

Fique atento ao remover uma atividade, pois ela é removida automaticamente ao clicar no botão Apagar esta atividade, sem caixa de confirmação.

Como adicionar uma unidade

Dentro de cada lição, você pode inserir quantas unidades desejar. As unidades contêm um vídeo e podem conter atividades.

Para adicionar uma unidade, realize os passos a seguir:

  1. Com acesso no papel de Admin, dentro do curso escolhido, acesse a lição em que deseja inserir uma ou mais unidades.
  2. A unidade 1 já vem por padrão em cada lição. Para adicionar mais unidades, clique no símbolo +.
  3. Adicione na unidade o vídeo (inserindo o link no campo Vídeo da Unidade) e, se quiser, atividades.
  4. Clique no botão Salvar.
  5. Clique no botão Publicar caso queira deixar as mudanças visíveis aos alunos.

Como apagar uma unidade

Você pode apagar qualquer unidade inserida.

Lembre-se que apagando a unidade, você está apagando também o vídeo e as atividades contidas nela. Para apagar, realize os seguintes passos:

  1. Com acesso no papel de Admin, dentro do curso, acesse a lição em que a unidade escolhida encontra-se.
  2. Clique na unidade.
  3. Clique no botão Apagar esta unidade.
  4. Clique em OK na caixa de confirmação que aparecerá.

Pronto! A atividade foi removida de seu curso e da plataforma TIM Tec.
