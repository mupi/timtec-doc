Guia TIM Tec Administrador

Apenas ao usuário que é administrador da plataforma é permitido criar um curso.

Como criar um curso

Primeiramente, é necessário estar na área de administração da plataforma. Para isso, após efetuar seu login, clique no botão que contém seu nome de usuário e selecione ADMINISTRAÇÃO.

Após isso:

  1. Clique no botão NOVO CURSO localizado à direita da página.
  2. Clique em cada um dos campos para inserir os dados do curso e salve cada alteração. Veja aqui como preencher cada campo.
  3. Salve o curso clicando no botão SALVAR. Veja que, aqui, o curso ainda não está visível para o público, mas apenas para você.
  4. Torne o curso visível ao público na plataforma clicando no botão PUBLICAR.

Descrição dos campos da página inicial do curso

O preenchimento de todo campo é obrigatório. Caso algum campo não seja preenchido, ele será exibido vazio ao aluno.

Veja a seguir a descrição de cada um dos campos. Para preenchê-los, basta clicar sobre eles e editá-los.

  1. Nome do curso: preencha com o nome do curso. Veja que abaixo deste campo é exibido o link direto para o curso. Este link é gerado automaticamente pela plataforma.
  2. Vídeo ou Thumbnail: aqui você escolhe se a página inicial do curso terá um vídeo de apresentação ou uma imagem. Caso for um vídeo, clique no botão VIDEO, insira o ID do link do YouTube e clique em ADICIONAR. O ID é a parte final do vídeo que o identifica, por exemplo, no link https://www.youtube.com/watch?v=mIvLsglRZdQ o ID é mIvLsglRZdQ. Caso for uma imagem, clique em THUMBNAIL, escolha o arquivo da imagem e clique em Adicionar.
  3. Resumo: insira aqui um resumo apresentando o curso. Nossa sugestão é que sejam usadas palavras-chave indicando sobre o que trata o curso.
  4. ADICIONAR INSTRUTOR: adicione o instrutor que será o responsável pelo curso. Você pode adicionar mais de um instrutor.
  5. Por que fazer esse curso?: descreva aqui de maneira clara quais são as aplicações dos conhecimentos adquiridos após a conclusão do curso, nos contextos profissional e social do aluno. Pense nas seguintes perguntas: o que posso estudar agora? O que consigo fazer agora? Em que posso trabalhar agora?
  6. O que eu preciso saber?: utilize este campo para descrever dois tipos de requisitos:
    • Quais os conhecimentos prévios necessários para realizar este curso.
    • Quais os requisitos de infraestrutura para realizar este curso.
  7. Estrutura: neste campo, explique de maneira sucinta a metodologia do curso, como serão oferecidas as aulas, quais os tipos de atividade, avaliação, enfim, quais os recursos de aprendizagens disponíveis.
  8. Aulas: este campo contém o principal conteúdo do curso. Aqui, serão inseridas as videoaulas e os vídeos. Para mais detalhes sobre o conteúdo das aulas clique aqui.
